Project Details

This page explores the creation of the Know Your Neighbourhood application, from inception to development and final implementation.

The Problem

Making Changes for a Better Future

The Toronto Police Service currently provides a large amount of data through a public web portal. Unfortunately, it is scattered across several web maps and dashboards and is hard to consume. The average person would find it hard to understand what is happening in their own neighbourhood. We wanted to fix that.

Project Scope

How Can We Help?

We wanted to build an easy-to-use web application that allowed people to understand crime in their neighbourhood. People should be able to tell what types of crime occur in their neighbourhood, whether crime is increasing or decreasing, and how they compare to the rest of the city.

Prepping the Data

FME Section

All of the data used in this project are from public sources. Using FME to connect directly to REST services and a file geodatabase, several distinct datasets were transformed and combined into a single occurrence layer. This consisted of all major crimes which occurred in Toronto between 2014 and 2020. This layer was then loaded to the Fleming College AGOL Organization as a hosted feature layer. This was the environment where the KYN Application was built.

Custom Point Symbology

Making super swanky symbols

Custom point symbology was created for all crime types that were used in the KYN application. We initially created a series of sketches during a brain-storming session (some stranger than others) and worked as a team to select those that we felt were the best fit for our application. These were then refined over the course of the project through consultation with the TPS. Colour was then applied that ensured we met the Web Content Accessibility Guideline (WCAG) standard for non-text contrast against our application basemap.
