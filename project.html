Project Details

This page explores the creation of the Know Your Neighbourhood application, from inception to development and final implementation.

The Problem

Making Changes for a Better Future

The Toronto Police Service (TPS) recognized in their 2017 review Action Plan: The Way Forward that current policing strategies require revision to better serve communities. A component of this initiative is to update to how policing data are consumed by the TPS and presented to the public.

The emphasis on both providing open data and shifting to community-centric policing has led the TPS to the decision to develop a web solution called Know Your Neighbourhood (KYN), which is intended to be a one-stop-shop for public safety data. The current TPS data portal is spread across several web map applications and has a city-wide focus. This results in an end-user experience that is negatively impacted by the presentation of relevant public safety data in separate places.

Project Scope

How Can We Help?

As a community partner of this project, Team EDGE will be assisting the TPS in developing the solution. In developing a solution, our primary objectives are:

1. Create a web map solution to display open crime data to the public, as well as additional functionality for skilled users to customize how they interact with the data.

2. Develop tools that will allow public users to view summarized statistics for neighborhood and police divisions within the city of Toronto.

3. Develop a set of design standards, including symbology and cartographic conventions, while adhering to AODA guidelines.

4. Create a user guide (for internal use) so that the TPS can reproduce our implementation as new needs arise.
